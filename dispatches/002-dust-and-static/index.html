Dust and Static

DISPATCH 002 // 2026.03.20 // FIELD CORRESPONDENT: M. OKAFOR

Okafor hadn't slept in thirty hours when the communications array in sector 7 started screaming. Not literally — the array doesn't have speakers. But the diagnostic panel lit up like a festival, every channel indicator spiking simultaneously, and the pattern buffer started dumping data so fast the local storage flagged a capacity warning. Something out there was broadcasting on every frequency the array could receive, all at once, and it had been doing it long enough to fill a terabyte of raw signal data before anyone noticed.

The sector 7 array is old hardware — pre-consolidation, bolted to a ridge overlooking the dust flats where the atmosphere thins out to almost nothing. It was scheduled for decommission twice and survived both rounds through bureaucratic oversight. Nobody monitors it in real-time. Okafor was only there because a routine maintenance flag had been sitting in his queue for two weeks, and his supervisor finally noticed. He'd driven four hours across the flats to swap a corroded antenna coupling. Instead, he found the array had captured something.

Pattern analysis on the captured data is still running. Preliminary results suggest the signal isn't random — there are repeating structures embedded in the noise, like someone buried a message inside a sandstorm. The origin point, based on the array's directional data, is somewhere deep in the dust flats. Somewhere no infrastructure exists. Somewhere nobody goes. Okafor has requested a survey drone deployment, but sector 7 requests are low priority. The data sits on a local drive, waiting. The array is still recording.

End of Dispatch
